# `annexe`

A general, modular Typescript utilities library for your everyday coding.

## Installation

To install annexe:

### Yarn

```sh
yarn add annexe
```

### NPM

```sh
npm install annexe
```

## Modules

The following modules are available to be installed independently

- @annexe/isfloat
- @annexe/isnumber
- @annexe/isstring
