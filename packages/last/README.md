# `@annexe/last`

A function to get the last element in an array.

## Installation

To install @annexe/last:

### Yarn

```sh
yarn add @annexe/last
```

### NPM

```sh
npm install @annexe/last
```

## Usage

```ts
import { last } from '@annexe/last';

const myArray = [1, 2, 3, 4, 5];
console.log(last(myArray)); // prints 5
```
